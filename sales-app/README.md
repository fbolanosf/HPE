# HPE Sales Specialist Toolkit (Next.js)

## Overview
A professional web application for HPE Sales Specialists to perform GAP analysis, financial modeling (TCO/ROI), and solution comparison.

## Tech Stack
- **Framework**: Next.js 13+ (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Charts**: Chart.js
- **Reports**: `docx` for Word export

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure
- `app/`: Application routes and pages.
- `components/`: Reusable UI components.
- `lib/`: Utility functions (calculations, exports).
- `public/`: Static assets.

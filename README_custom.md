# HPE Sales Specialist Toolkit

## Overview
This application is designed for HPE Virtualization Sales Specialists to streamline the sales process, assess client needs, and demonstrate the financial and technical value of HPE solutions (GreenLake, Zerto, Hybrid Cloud).

## Key Modules
1.  **GAP Analysis**: Assess current infrastructure against business goals.
2.  **Financial Modeling**: TCO/ROI calculators for migration to HPE GreenLake/Hybrid Cloud.
3.  **Solution Sizing**: Tools to size and compare virtualization and data protection solutions.
4.  **Reporting**: Professional PDF generation for client presentations.

## Technology Stack
- **Backend/Frontend**: Python + Streamlit (Proposed) for rapid, data-centric development.
- **Data Storage**: Local JSON/SQLite for assessment data.

## Usage
Run the application with:
```bash
streamlit run app.py
```
